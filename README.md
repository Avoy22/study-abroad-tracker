# Study Abroad Tracker

Phase 1 foundation for a full-stack Study Abroad Tracker application.

## Structure

```text
study-abroad-tracker/
  client/   React + TypeScript + Vite frontend
  server/   Express + TypeScript backend
```

## Phase 1 Scope

- Create the monorepo folder structure.
- Configure the frontend with React, TypeScript, Vite, and Tailwind CSS.
- Configure the backend with Express, TypeScript, and baseline middleware.
- Add environment examples, useful npm scripts, and ignore rules.

Authentication, CRUD features, and frontend-to-backend integration are intentionally not implemented yet.

## Local Development

Install dependencies inside each app folder, then run each service in its own terminal.

```bash
cd client
npm install
npm run dev
```

```bash
cd server
npm install
npm run dev
```
